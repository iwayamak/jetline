# S3 モジュール定義

## 1. `S3`（mode: `Put`）

ローカルファイルを S3 へアップロードします。

### 必須パラメータ

- `s3_component_key` (str)
- `local_file_path` (str)
- `s3_dir_path` (str)

### 任意パラメータ

- `end_file_name` (str, default: `null`)

### 例

```yaml
sub_module:
  - name: S3
    mode: Put
    param:
      s3_component_key: S3_COMPONENT.ID=UT
      local_file_path: data/*.csv
      s3_dir_path: inbound/daily
      end_file_name: END
```

## 2. `S3`（mode: `Get`）

S3 オブジェクトをローカルへ取得します。

### 必須パラメータ

- `s3_component_key` (str)
- `s3_file_path` (str)
- `local_dir_path` (str)

### 任意パラメータ

- `end_file_name` (str, default: `null`)

### 例

```yaml
sub_module:
  - name: S3
    mode: Get
    param:
      s3_component_key: S3_COMPONENT.ID=UT
      s3_file_path: inbound/daily/*.csv
      local_dir_path: tmp/download
```

## 3. `S3`（mode: `List`）

S3 上のオブジェクト一覧を取得します。

### 必須パラメータ

- `s3_component_key` (str)
- `s3_file_path` (str)

### 例

```yaml
sub_module:
  - name: S3
    mode: List
    param:
      s3_component_key: S3_COMPONENT.ID=UT
      s3_file_path: inbound/daily/*.csv
```
